Frequently Asked Questions

How does the website analysis engine work?
SEO Vision launches an isolated Chrome container in headless mode using Puppeteer. It loads the target URL, waits for all Javascript frameworks (like React, Angular, or Vue) to render, and captures the final generated HTML. This HTML is parsed using Cheerio to evaluate tags, contents, and schemas.
Do I need to pay for Google Lighthouse APIs?
No! The platform operates entirely on open-source libraries. We measure connection performance, DOM sizes, and semantic SEO parameters directly using custom modules without relying on any paid third-party APIs.
Are PDF downloads styled?
Yes! The PDF generation tool runs Chrome's native print engine in the background to capture the report dashboard with exact CSS variables, colors, tables, and glassmorphic panels.
